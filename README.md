⚙️ Quick Start Guide
1. Installation

Clone or extract the project, then run the following command in your terminal to install all necessary dependencies:
Bash

pip install -r requirements.txt

2. Launching the Engine

For a seamless experience, run the dashboard directly:
Bash

streamlit run app.py

3. Usage

    Open the provided local URL (usually http://localhost:8501) in your browser.

    Upload your CSV file (Ensure it contains ds and y columns).

    Use the sidebar to adjust market scenarios.

    Click "Generate Enterprise Forecast" to view your predictive analytics.
